序列，字符串，元祖，及字典

列表

索引

可以通过索引来获取，删除，修改
del list[0]
list[0] =1

list = [‘a’,’b’,’c’,’d’]
通过索引获取 list[0] ,输出 a ,如果获取一个超出范围的索引会提示错误。可以通过负数来获取从右边开始的值。
list[-1] 获取的是倒数第一个

分片

索引只能操作单个元素，有时候需要操作某个范围的元素.使用: 冒号来操作索引范围
list[1:2] 标示从下标1开始，获取到下标2范围的值。如果范围直接没有值则返回空的列表
list[1:1]
这个也可以使用负数，来从右边开始获取
list[-2:-1]

list[-2:] 右边这个下标省略，标示一直获取到末尾，也可以用于开始下标
list[:3] 获取从0开始到3
通过这个可以用来复制整个列表，把两边下标值为空 list[]

列表推导式
快速获取一个100以内偶数的列表
list2 = [x for x in range(101)]
切边之步长,及第2个冒号后的值，这个值默认为1 ，如果设置成大于1时，会跳过一些数
list2 = list2[0:100:2]

以上的方法来获取偶数列表比较麻烦，以下直接通过推导式就可以了
list2=[x for x in range(101) if x%2==0]

可以写多个循环，甚至一个语句写出99乘法表

a = ‘\n’.join([‘ ‘.join([‘%s%s=%s’ % (x,y,xy) for x in range(1,y+1)]) for y in range(1,10)])

print(a)
这个虽然只是写到一起，但也可以看出推导式的作用是用来生成列表的。

序列相加，相乘

list = [‘a’,’b’]

list2 = [‘b’,’c’]

list+list2 // 会组成新的列表 [‘a’, ‘b’, ‘b’, ‘c’]

list * 3 // 会重复三次 [‘a’, ‘b’, ‘a’, ‘b’, ‘a’, ‘b’]

常用列表内建函数

len(list) 获取列表长度，同样可用于获取字符串长度

max(list) 获取最大的元素

min(list) 获取最小的元素

方法
x.append(e) 在列表末尾添加

x.count(e) 统计e出现的次数，也可以用于字符串

x.index(e) 出现的下标位置

x.insert(index,e) 指定下标插入

x.pop() 删除并且返回

x.remove(e) 删除

x.sort() 排序

字符串

字符串是程序最主要的构成，Python也提供了如何更好的操作字符串。
对序列操作的索引，分片，乘法，max,min ,len(),count都对字符串有作用。但是要主要分片赋值对字符串是不起作用的，字符串本质是不可变的。

字符串格式化 ，这个同Java格式化字符串区别不大
s = ‘hello %s’
print(s % ‘world’)

s= ‘ %s %d,%s %f’
params = (‘价格是：’ ,100,’打折后：’ ,98.2)
print(s % params)

如果这个不能满足要求，可以使用模版字符串 Template ,类似freemarker的参数变量替换，当然功能没ftl强大。

字符串方法，很多有用的方法。

find(e,start,end) 查找，并且返回开始位置的索引，没找到-1,后面两个是可选参数，指定搜索范围。

join() ，前面乘法表用过，把一个列表内的元素相加，元素必须是字符串
‘,’.join([‘1’,’2’]) split()的逆方法

replace () 查找并替换

split() 切割字符串返回列表

strip() 去除首尾空格

…. 省略其他

del map[‘key1’] //删除

map[‘key1’] = ‘test’ //更新

len(map) 键值对数量

‘key1’ in map //检查是否存在这个键

clear

get(k,default) //通过key获取值，无则默认返回default

has_key(k) // k 是否存在

keys() 返回列表，以键的形式

pop(k) 获取k的value，不起从字典移除

setDefault(‘n’,f)设置值

values() 与keys()

update(dict) 以一个字典的值更新另一个字典
